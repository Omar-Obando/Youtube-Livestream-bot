```bash

```

1. Run the bot:
```bash
python YouTube-Livestream-bot.py
```

2. Select language (1: English, 2: Español)

3. Enter YouTube stream ID

4. Configure session duration:
   - Indefinite

5. Configure number of windows:
   - Default (10 windows)
   - Custom (1-50 windows)

6. Select proxy loading method:
